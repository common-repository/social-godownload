This plugin is a Social Download Manager.  For download first you have to share it at least in one social network.

== Description ==

Wordpress Plugin that works as a social download manager, ie allows downloading a file only if they are shared by a social network (Twitter, Facebook or Google Plus). The version FREE only contain Twitter button.

== Installation ==

1. Upload `Social-GoDownload.zip` to the `/wp-content/plugins/` directory
2. Unzip the plugin
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Place shortcode `[godow]` in your templates

== Frequently Asked Questions ==

= How can I display only a category? =

You can use the shortcode [godow cat=num-cat] where num-cat is a category's key

= I have problems with viewing the Facebook. =

Maybe you have deleted this code of the textarea (style) at settings page

/* DONT MODIFY */
.fb-like span,.fb-like, iframe.fb_ltr {
position: relative;
float: left;
min-width: 450px;
min-height: 61px;
}
